# Simple Smart Contract

Super simple smart contract example

## Run

```bash
# run test in test directory
truffle test

# start local blockchain using Ganache
truffle develop

# compile and deploy (with clearing existing contracts)
truffle(develop)> migrate --reset
# this will create build directory

# deploy details is in build/contracts/SimpleSmartContract.json
# abi: application binary interface (json object)
# abi shows externally accessible functions of the smart contract

# start static server
npm start
open http://localhost:9080/

# check console
# Contract: methods is important
# Array: list of account generated by Ganache
```
